1-AO ENTRAR NO MLINUX 

    Logo que entrar no Mlinux ele vai dar uma mensagem dizendo que é a 
primeira vez que você entra no sistema (logo depois que vc descompactar os 
arquivos e executar Linux no diretório c:\linux). 
Será dado o login. Para quem ainda não conhece o linux possui um sistema de 
"contas" que cadastram os usuários para terem acesso ao sistema. É como se 
você chegasse no trabalho e eles pedissem a sua identificação através de seu 
nome (login) e senha (pass). 
    Digite root e tecle . Depois disso você deverá digitar passwd 
para configurar sua senha. Tecle a senha (password) e . Será 
necessário redigitar a senha. Tecle enter. Obs.: lembre-se que o linux é 
"case sensitive", e isto quer dizer que nomes em maiúsculo são diferentes 
de nomes em minúsculo, ou seja, DOS é diferente de dos, diferente de DoS e 
de dOs. Assim se você escolher senha98 como password será diferente de 
SENHA98. Por questões de segurança não use senhas óbvias como root ou user, 
procurando misturar maiúscula com minúscula e números. Ex.: Bakurih Grupo de
Estudo -> login: bakurih senha: BaKeStud98. 
    Por questão de conveniência cadastre um outro usuário para você utilizar
o sistema, prevenindo assim que você faça alguma besteira usando a conta de 
root, pois o root tem "permição" para tudo, até mesmo apagar arquivos de 
sistema que não deveriam ser apagados. Com uma conta normal o linux previne 
isso (não é uma maravilha). Mais para frente explicaremos as permissões e 
tudo relacionado a elas. 

2-ADICIONANDO UM USUÁRIO 
    Para adicionar um usuário digite adduser  . Após isso será apresentado 
a seguinte msg: 
Please answer the following questions about the new user added. 
(o que estiver entre colchetes é padrão, digite enter) 

Login:   
Real Name: Bakunin 
Group: [users] 
Other groups: [none] 
Home directory:[/home/bakurih] 
Shell: [/bin/tcsh] 
uid: [first free] 
Expire date of this account (mm/dd/yy): [never] 
Maximum days between password changes: [180] 
Minimum days between password changes: [0] 
Number of warning days before password expires: [14] 
Number of usable days after expiration: [14] 
Set password:   

Redigite o pasword e pronto...
