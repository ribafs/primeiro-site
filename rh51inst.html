Parte 1 - Criando um disco de BOOT :

Para criar os discos vocÊ precisará do CD do Linux Red Hat e de um disquete formatado.

No prompt do DOS, na unidade de CDROM, digite ezstart.
ATENÇÃO : Para rodar o ezstart, você não poderá estar no prompt do Windows e sim no modo MS-DOS.

Digite y para a pergunta : Are you sure are not running under a shell ?

Entre com a letra do seu drive de CDROM.

Escolha a opção B para o método de instalação.

Insira o disco no drive e aguarde a conclusão da criação do disco.

Após criado, reinicie seu computador com o disco inserido no drive.

Parte 2 - Instalando o Linux :

Após a tela Welcome to Red Hat Linux aparecer, serão feitas perguntas simples sobre o sistema. Na opção de teclado, escolha US.

Em installation Method, escolha Local CDROM.

Em SCSI Configuration, escolha não, exceto se você possuir um adaptador SCSI.

Em Installation Path, escolha Install.

Em Disk Setup, escolha Fdisk para poder criar as partições Linux.

Em Partition Disks, selecione o HD e depois escolha EDIT.

Para saber mais sobre o funcionamento do FDISK do Linux, clique aqui.

Em components to Install, escolha os pacotes que deseja usar. Após isso, o linux será copiado para seu computador.

Em Proibing Result, o seu mouse será detectado automaticamente.

Em Configure Mouse, você escolherá o tipo de mouse mais adequado.

PCI probe, irá detectar seu hardware.

Network Configuration, para configurar sua rede local.

Configure Time Zones, escolha do fuso horário.

Configure Printer, configuração de uma impressora local ou em rede.

Root Password : A senha para o principal usuário do Linux (deve ter mais de 6 caracteres).

LILO Instalation : o LILO é o programa usado para se escolher, no momento do BOOT, qual o sistema que será carregado.

Terminada a instalação, você pode agora reiniciar seu computador, agora sem o disco de BOOT no drive
