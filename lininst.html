1 Primeiros Passos
Este manual explica passo-a-passo todos os procedimentos de instalacao do unix (LINUX) em particao estendida do HD, ou seja nao explica o procedimento de instalacao em cima do MSDOS (na particao primaria), que por sua vez nao eh aconselhavel.

O linux roda em maquinas 386SX e superior o minimo de memoria RAM eh 4Mb e tambem roda c/ monitor CGA.
ps: usaremos apenas disquetes e arquivos de 1,44Mb

O Manual do linux completo (em ingles) vc encontra neste site: (sunsite.unc.edu:/pub/Linux/) tambem neste site vc encontra todo tipo de documentacao sobre o sistema, deste FAQ, HOWTO ate DOC em geral.

Para voce continuar lendo este manual, voce devera possuir os seguintes requisitos minimos. Saber particionar o HD, pois conheco pessoas, que tem muita vontade em fucar no computador, mas ainda nao sabe fazer este processo de particionar. Vale a pena lembrar, que o procedimento usado pelo o Linux *nao* eh igual ao do MSDOS, e eh mais complicado. mas se mesmo assim vc quiser arriscar va em frente e boa sorte, pois eh perdendo os dados que a gente aprende. :)

Computador usado como exemplo: PC-AT 486DX2 Intel - Winchester IDE

Espaco livre minimo: 15Mb (voce nao ira conseguir instalar nenhum pacote!)
Espaco livre recomendado, entre 150 a 250Mb

3 Instalando Apenas o Básico
Para iniciar voce precisa copiar +- 6Mb de arquivos, que sao: bare.gz / color144.gz / rawrite.exe / gzip.exe e os diretorios A1 / A2 / A3 / A4

ps: quando vc for baixar os arq. note q tera dois arquivo bare e
color144 copie apenas os da versao de 1,44Mb q sao os de 5??Kb
 
Arquivo/Diretório Descrição
gzip descompacta arq. com extensao .gz
bare.gz. cria disco de boot
color144.gz cria disco de root
rawrite.exe formata o disquete com os arquivo acima
DIR (A1-A2-A3-A4) arquivos basicos para rodar o linux
Depois de ter feito a copia vc precisara de 6 disquetes (1,44Mb).
No diretorio q vc baixou os arq., digite o seguinte:

gzip -d bare.gz
gzip -d color144.gz

Insira o disquete e digite...

rawrite bare a:

troque o disquete e digite...

rawrite color144 a:

depois copie cada A? para os 4 disquetes.

copy A1\*.* A:

Pois bem o disquete q vc copiou o bare eh o disco de BOOT e o colo144 o de ROOT. Agora resete a maquina e insira o disco de BOOT.

ps1: caso vc tenha dados na particao primaria do HD, nao se preocupe pois o linux nao apagara.
ps2: faca backup sempre.
 
 
4 Particionando o HD
Depois do boot siga as instrucoes que apareceram na tela.

Logo apos o boot com disquete ira aparecer...

boot:
tecle e enter, entao aparecera mais mensagens, e em seguida sera solicitado trocar o disco pelo o do ROOT, faca isto e tecle enter. Aguarde alguns segundo e aparecera uma mensagem dizendo para digitar 'root' e entao caira o shell do unix. Ai que a historia comeca a ficar boa.

O Linux pede que depois de digitar 'root' digite 'setup' para comecar a configura-lo, mas antes, este procedimento so ira adiante se seu winchester ja estiver particionado.

slackware login: root
#
# setup
erro, digite primeiro 'fdisk'

# fdisk

Lembre-se sua unidade C e chamada pelo linux d '/dev/hda' ¤ esqueca disso! para criar a particao e barbada, e so seguir os exemplos. a unica diferenca e a capacidade do seu HD com a do exemplo.

digite '?' para ver os camandos do fdisk. Use o comando 'p' para ver as informacoes (particoes) atuais.

Comecando...

Primeiro use o comando 'p' par ver a(s) particao(oes) corrente. Se voce ja possui uma particao primaria no DOS, note que aparecera... veja o exemplo:

Cuidado: para nao deletar sua particao primaria DOS/Win, o camando para deletar  'd' mais a particao que no caso  o numero '1'

Proximo passo - Use o comando ``n'' para criar a nova particao. Vamos supor que voce queira deixar 80Mb para o linux.

Partition number (1-4): 2
First cylinder (204-683): 204
Last cylinder or +size or +sizeM or +sizeK (204-683): +80M

O linux mostrou que tem (204-683). Voce tem que informar o primeiro numero '204' depois na outra linha tem que informar o numero de MBytes para a particao ex. '+80M' Ai eh sera necessario criar um particao 'virtual' ... siga o exemplo

Command (m for help): n
Command action
e extended
p primary partition (1-4)
p

Partition number (1-4): 3
First cylinder (474-683): 474
Last cylinder or +size or +sizeM or +sizeK (474-683): +10M

Ficara mais ou menos assim:

Command (m for help): p
Disk /dev/hda: 16 heads, 38 sectors, 683 cylinders
Units = cylinders of 608 * 512 bytes

Device Boot Begin Start End Blocks Id System
/dev/hda1 * 1 1 203 61693 6 DOS 16-bit >=32M
/dev/hda2 204 204 473 82080 83 Linux native
/dev/hda3 474 474 507 10336 83 Linux native

Voce precisa trocar a particao /dev/hda3 de 'Linux native' para 'Linux Swap' use o comando 't'...

Command (m for help): t
Partition number (1-4): 3
Hex code (type L to list codes): 82

Agora voce precisa 'escrever' <w>rite no winchester, confira mais uma vez a particao teclando 'p' se estiver tudo legal (parecido com o exemplo) tecle 'w' para gravar ou q de quit para sair sem gravar.

Depois  so rebootar a maquina e fazer o procedimento de boot e root e quando o disco de root solicitar que digite 'setup', siga a instalacao. pois seu winchester ja esta particionado.
 
 
5 Instalando os Pacotes (via disquete)
Primeiro voce tem que instalar o basico, os disquetes que contem os arquivo/diretorios A1 A2 A3 A4, para fazer isto utilize o menu SOURCE. depois de instala-los va para o menu CONFIGURE. para configurar o linux.

Para instalar os demais pacotes use o menu PKGTOOL
